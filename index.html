This software allows you to move files between computers on your local network. It runs on Windows and gives you control over who sees your files. You manage access through users and groups while the app keeps an audit log of all transfers.

📦 Features

  • Network Speed: Sends files directly over your local Wi-Fi or Ethernet connection.
  • User Access: Creates specific accounts so you choose who accesses your shared folders.
  • Group Permissions: Groups users together to manage set permissions for multiple people at once.
  • Transfer History: Records all file movements in a clear log file for your review.
  • Simple Settings: Uses a standard windowed display for easy navigation.

⚙️ System Requirements

To run this application, your computer needs the following:

  • Windows 10 or Windows 11.
  • An active network connection (Wi-Fi or Ethernet).
  • At least 100 MB of free storage space.
  • Standard administrator permissions to install software.

👤 Using the Application

When you open the app, you will see a main dashboard. This dashboard shows your active network connections. To start sharing, navigate to the settings tab. Here you can create a user account. Assign this account to a group if you have many users to manage. Once you designate a folder as shared, other people on your network will see it based on the permissions you set. You can check the activity log button at any time to see a list of who accessed your files and when.

🔒 Security

The application stores user credentials locally on your computer. It does not send your data to the internet. All file transfers stay within your local network. This keeps your sensitive files private. The audit log provides a record of every file operation. Review this log if you suspect unauthorized access.

🆘 Frequent Questions

The app does not detect other computers. Check if both computers connect to the same router or network switch. Ensure your firewall allows the app to communicate.

I forgot my password. You can reset user accounts through the administrator settings menu.

Is the transfer speed slow? Wired Ethernet connections perform better than Wi-Fi. Check your cables if you experience slow speeds.
